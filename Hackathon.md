# 0G APAC Hackathon

Build the next generation of AI x Web3 applications on 0G’s modular infrastructure.

---

## Overview

The 0G APAC Hackathon is a builder program for developers creating AI x Web3 applications using 0G’s modular infrastructure.

The hackathon runs from March to May 2026 and supports developers, founders, and early-stage teams building across:

- Autonomous agents
- Verifiable finance
- AI-native applications
- Privacy infrastructure
- High-performance consumer products

Participants receive technical support, ecosystem exposure, and opportunities within the 0G ecosystem.

# Submission Requirements

## 1. Project Information

Each submission must include:
- Project name
- One-sentence description (max 30 words)
- Problem statement
- Explanation of 0G integrations used

---

## 2. Code Repository

Requirements:
- Public GitHub repository
- Meaningful development activity during the hackathon
- Empty or placeholder repositories may be disqualified

---

## 3. 0G Integration Proof

Mandatory requirements:
- 0G mainnet contract address
- 0G Explorer link
- Proof of 0G integration

### Accepted Components
- 0G Storage
- 0G Compute
- 0G Chain
- Agent ID
- Privacy & secure execution features

Projects without real 0G integration are invalid.

---

## 4. Demo Video

Requirements:
- Maximum 3 minutes
- Must demonstrate:
  - Core functionality
  - User flow
  - Actual 0G usage

### Accepted Platforms
- YouTube
- Loom
- Publicly accessible links

Slide-only videos are not accepted.

---

## 5. README / Documentation

Must include:
- Project overview
- Architecture diagram or technical explanation
- 0G modules used
- Deployment instructions
- Testing instructions
- Reviewer notes if needed

Strong documentation is part of judging.

---

## 6. Public X Post

Each team must publish at least one public post on X.

### Required Content
- Project name
- Demo screenshot or clip

### Required Hashtags
- #0GHackathon
- #BuildOn0G

### Required Tags
- @0G_labs
- @0g_CN
- @0g_Eco
- @HackQuest_

This requirement is mandatory.

## 7. Optional Bonus Materials
